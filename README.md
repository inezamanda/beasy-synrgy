## Beasy REST API
Beasy is dummy banking app with gamification.

Swagger Api Documentation : [SWAGGER](https://beasy-api-dev.herokuapp.com/swagger-ui.html)

## Technology Stack
- Java
- Spring Boot
- Spring MVC
- Spring Security
- Spring Data JPA
- PostgreSQL
- Cloudinary
